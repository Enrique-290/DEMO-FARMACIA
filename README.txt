Suite Salud — DEMO local (Farmacia + Consultorio + Laboratorio)
1) Abre index.html en Chrome/Edge.
2) Todo se guarda localmente (localStorage e IndexedDB). No requiere internet.
3) Exporta CSV desde cada módulo; imprime tickets y notas desde el navegador.
4) Limpia datos en Configuración → Mantenimiento (útil para equipos de prueba).
